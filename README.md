# 🩺 Control de Presión Arterial

Aplicación web diseñada para registrar, monitorear y llevar el control de la presión arterial de forma sencilla, rápida y segura.

---

## 📌 Descripción

Esta app permite guardar datos de presión arterial (sistólica y diastólica), frecuencia cardíaca y observaciones, ayudando a llevar un seguimiento diario del estado de salud.

Ideal para uso personal o familiar.

---

## 🚀 Funcionalidades

- 📊 Registro de presión arterial (sistólica / diastólica)
- ❤️ Registro de frecuencia cardíaca
- 📝 Notas u observaciones por registro
- 📅 Historial de mediciones
- 🔍 Búsqueda y filtrado de datos
- 📈 Visualización del progreso
- 💾 Almacenamiento local (sin necesidad de internet)
- 📱 Compatible como aplicación (PWA)

---

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- LocalStorage (almacenamiento en el navegador)
